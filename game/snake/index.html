🍎 貪食蛇

《貪食蛇》是一款經典的小遊戲

玩家需要控制小蛇在地圖中移動，尋找並吃掉出現的食物。 每吃到一顆食物，小蛇的身體就會變長，速度也會逐漸提升。 隨著遊戲進行，操控難度會越來越高，玩家必須靈活轉向，避免撞到牆壁或自己的身體。

隨著分數不斷提高，小蛇的移動速度會逐步加快，挑戰玩家的反應能力與路線規劃。 你的目標是讓小蛇不斷成長，創造更高的分數紀錄。

遊戲操作：

W / ↑ ：向上移動

S / ↓ ：向下移動

A / ← ：向左移動

D / → ：向右移動

地圖模式：

⚡小地圖：

小尺寸地圖，空間較小，速度較快，適合高手挑戰。

🎯經典圖：

標準尺寸地圖，節奏平衡，適合一般遊玩。

🌍大地圖：

大型地圖，空間較大，速度較慢，適合新手練習。

遊戲目標：

吃到更多食物，讓小蛇不斷成長並獲得更高分數。

挑戰不同地圖模式的最高紀錄，在速度不斷提升的情況下存活更久。
