渡劫修仙

《渡劫修仙》是一款修仙主題的免費網頁小遊戲。

玩家需要左右移動閃避落雷，在雷劫中存活並累積突破值。當突破值滿時即可突破境界，並選擇新的天賦能力。 隨著境界提升，雷劫會變得更加猛烈，玩家必須合理搭配天賦與道途，才能在天劫之中存活更久。 你的目標是突破更高境界，挑戰更高分數。

遊戲操作：

A / ← ：向左移動

D / → ：向右移動

境界系統：

練氣、築基、金丹、元嬰、化神。修仙者會隨著突破逐步提升境界。

天賦系統：

護體：增加護盾值

回氣：生命恢復速度提升

罡氣：減少雷劫傷害

悟性：突破值累積速度增加

煉血：提升最大生命值

凝神：提升突破穩定度

道途系統：

玄武金身道：

條件：鐵骨 Lv.3 + 罡氣 Lv.2

效果：大幅減少雷劫傷害

長生回春道：

條件：回氣 Lv.3 + 護心 Lv.2

效果：持續恢復生命

通明悟道道：

條件：悟性 Lv.3 + 凝神 Lv.2

效果：突破速度提升

血煉化劫道：

條件：煉血 Lv.3 + 金身 Lv.2

效果：雷劫傷害部分轉化為生命

遊戲目標：

在雷劫中存活更久，挑戰更高的分數紀錄

突破更多境界，解鎖更強的天賦與道途

每一局的天賦選擇不同，道途也會不同

每一次渡劫都是全新的修仙體驗
